Pacman in VHDL

VHDL implementation of retro game Pacman using various capabilities of Xilinx developments board Basys 3 such as VGA display port and seven segment displays. Project serves as a proof of concept for recreating the game as Basys3 is a resource constraint environment, for project to be scalable functions and subroutines are defined and implemented to be independent of resolution and platform.

VatanScrapper

Python application that uses Beautiful Soup 4 for parsing web pages into useful data, specifically this application focuses around gathering data such as price and specs of products from Vatan Computer’s website a famous electronics store in Turkey.

DailySpokesMan

The name of the application comes from a popular newspaper company Sözcü whose name roughly translates to spokesman. This application uses the core utilities developed for VatanScraper for gathering data and Flask web development micro framework and Bootstrap 4 generating a front-end for newly generated news pages and for data storage uses SQL-Alchemy a wrapper package around various SQL implementations.
